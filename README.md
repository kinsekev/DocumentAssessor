# Document Assessor Project

## Setup

```
Technologies:

install NodeJS
install MongoDB
```

```
Spin up mongo by entering 'mongod' into a terminal
Run git clone https://github.com/kinsekev/DocumentAssessor.git (or clone your own fork of the repository)
Go into the cloned folder with cd DocumentAssessor
Run npm install (only has to be done before first start or when you change the source code)
Run npm start
Browse to http://localhost:3000
```

```
To create assessment:

Register and login to the application
Click create assessment
Fill the form out (example text file is - links1.txt)
Submit
```
